# AARRR 增长漏斗模型

## 概述
AARRR 是 Dave McClure 提出的增长模型，又名"海盗模型"，帮助产品经理系统性思考用户增长路径。

## 五个阶段

### 1. 获取（Acquisition）
- 用户从哪些渠道发现你的产品？
- 关键指标：下载量、访问量、获客成本（CAC）
- PRD 关注：获客渠道设计、SEO/ASO 策略、裂变机制

### 2. 激活（Activation）
- 用户首次使用后是否体验到"Aha Moment"？
- 关键指标：注册转化率、首次关键行为完成率
- PRD 关注：新人引导流程（Onboarding）、核心价值快速触达

### 3. 留存（Retention）
- 用户是否会反复使用你的产品？
- 关键指标：次日/7日/30日留存率、流失率
- PRD 关注：推送通知策略、用户习惯养成机制、核心循环设计

### 4. 变现（Revenue）
- 如何从用户处获得收入？
- 关键指标：ARPU、LTV、付费转化率
- PRD 关注：商业模式设计、定价策略、付费点设计

### 5. 推荐（Referral）
- 用户是否愿意推荐给他人？
- 关键指标：NPS、邀请率、K-factor
- PRD 关注：邀请激励机制、社交分享功能、口碑传播设计

## PRD 中的应用
- 在"运营策略"章节按 AARRR 阶段规划关键措施
- 在"成功指标"章节列出各阶段的北极星指标
- 在"功能需求"中标注每个功能对应的增长阶段
- 确保产品方案覆盖完整增长链路
